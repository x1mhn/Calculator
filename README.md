# Mohnish Calculator

A clean, responsive web calculator built with HTML, Tailwind CSS, and vanilla JavaScript.

## Features

- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division, Modulo
- Keyboard support for quick input
- Clean, modern dark-mode UI
- Responsive design for all screen sizes
- Error handling (division by zero)

## Demo

Open `index.html` in any modern web browser to use the calculator.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Append number |
| `.` | Decimal point |
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `%` | Modulo |
| `Enter` or `=` | Calculate result |
| `Backspace` | Delete last digit |
| `Escape` | Clear all |
